## 社交媒体认证模板

####标准可信声明示例

```json
{
	"Context":"claim:twitter_authentication",
	"Id":"12312fdfvdf",
	"Claim":{
		"Name":"张三",
		"UserId":86076389,
		"Alias":"tony"
	},
	"Metadata":{
		"CreateTime":"2017-01-01T22:01:20Z",
		"Issuer":"did:ont:8uQhQMGzWxR8vw5P3UWH1j",
		"IssuerName":"onchain",
		"Subject":"did:ont:4XirzuHiNnTrwfjCMtBEJ6",
		"Expires":"2018-01-01"
	},
	"Signature":{
		"Format":"pgp",
		"Algorithm":"ECDSAwithSHA256",
		"Value": ""
	}
}
```

字段说明：

| Field     |     Type |   Description   | 
| :--------------: | :--------:| :------: |
|    Context|   String|  可信声明模板标识  |
|    Id|   String|  可信声明唯一标识id  |
|    Claim|   Object|  可信声明具体内容，key-value形式，自定义  |
|    Metadata.CreateTime|   String|  创建时间,格式：yyyyMMdd'T'HHmmss'Z'  |
|    Metadata.Issuer|   String|  可信声明颁发者ontid  |
|    Metadata.IssuerName|   String|  可信声明颁发者名称。该字段可以没有  |
|    Metadata.Subject|   String|  可信声明被颁发者ontid  |
|    Metadata.Expires|   String|  过期时间，格式：yyyyMMdd。该字段可以没有，表示该可信声明没有有效期  |
|    Signature.Format|   String |  签名格式  |
|    Signature.Algorithm|   String |  签名算法  |
|    Signature.Value|   String |  签名值  |
